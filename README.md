# PrivatBank Fullstack Application

## Overview
This is a full-fledged demo of a PrivatBank card application. It features a modern frontend card UI and a backend server for card data management.

## Features
- Interactive card flip animation
- Card details: number, holder, expiry, CVV
- Contactless and chip visuals
- Responsive design
- Backend API (Node.js/Express)

## Getting Started

### Prerequisites
- Node.js (v16+ recommended)

### Installation
1. Clone the repository:
   ```sh
   git clone <repo-url>
   cd Privatbank-fullstack
   ```
2. Install backend dependencies:
   ```sh
   cd backend
   npm install
   ```

### Running the Application
1. Start the backend server:
   ```sh
   node server.js
   ```
2. Open `frontend/index.html` in your browser.

## Project Structure
```
backend/      # Node.js backend (Express, routes, database)
frontend/     # Frontend static files (HTML, CSS, JS)
```
